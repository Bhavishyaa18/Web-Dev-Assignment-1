# 🚀 Dynamic To-Do List App

A sleek and minimal **Dynamic To-Do List Web App** built using **HTML, CSS, and JavaScript** — designed to help you stay organized, productive, and focused.

---

## ✨ Features

* ✅ Add new tasks instantly
* 📝 Edit existing tasks
* ❌ Delete tasks easily
* 📌 Mark tasks as completed
* 💾 Persistent storage (keeps your tasks even after refresh)
* 🎯 Clean and responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
* **Storage:** Local Storage (Browser-based)

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/bhavishyaa18/dynamic-to-do-list-app.git
```

### 2️⃣ Navigate to the project

```bash
cd dynamic-to-do-list-app/frontend
```

### 3️⃣ Run the app

Simply open `index.html` in your browser 🎉
